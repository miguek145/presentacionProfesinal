Blog of Notes

This application allows the user to save notes created by the user and can also create to-do lists and the tasks that are created in each list can be marked as completed tasks and there are tasks that do not belong to a list.

In this picture we can see the tasks that belong or not to a list and we can see their priority type, the title of the task list and the expiry date.

We can see the list of tasks created by the user and clicking on it will display the tasks of this list.

This is the page where the user will be able to create his notes and apply the colour he wants to each note and to see the content of each note the user will have to click on it.
